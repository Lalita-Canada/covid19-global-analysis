# COVID-19 Global Data Analysis - Key Insights

## Project Overview
Analyzed global COVID-19 data using Python to 
uncover key trends and insights across countries 
worldwide.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
1. USA had the highest confirmed COVID-19 cases globally
2. Global cases grew exponentially between 2020-2021
3. Germany had higher death rates than France and Spain
4. Some countries achieved recovery rates above 90%
5. Death rates varied significantly across countries 
   due to healthcare infrastructure differences

## Dataset
Novel Corona Virus 2019 Dataset from Kaggle
